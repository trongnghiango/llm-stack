#### Những gì đã thêm

#### Spec tool `Bash`

```
Tên tool : Bash
Tham số  :
  - command  (string, bắt buộc) – lệnh bash cần chạy
  - timeout  (number, tuỳ chọn) – giây, mặc định 30, tối đa 120
  - cwd      (string, tuỳ chọn) – working directory, mặc định = thư mục binary proxy
Output   : stdout + stderr gộp chung, tối đa 32 KB
Exit != 0: KHÔNG trả lỗi hard, trả về "Exit code: N\n--- Output ---\n..."
Timeout  : trả về "Command timed out after Ns.\n--- Partial output ---\n..."
```
