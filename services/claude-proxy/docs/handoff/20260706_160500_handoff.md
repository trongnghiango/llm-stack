## 3. Các bước xác minh & chạy thử
Để kiểm tra tính toàn vẹn của hệ thống, lập trình viên tiếp theo có thể chạy các lệnh sau:

```bash
# 1. Chạy bộ unit test để đảm bảo không phát sinh lỗi logic mới
go test -v ./...

# 2. Thực hiện build proxy để kiểm tra biên dịch thành công
go build -o proxy .

# 3. Khởi chạy thử proxy và kiểm tra logs được ghi đúng định dạng
./proxy
```
