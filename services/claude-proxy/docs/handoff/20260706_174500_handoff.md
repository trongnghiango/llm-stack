## 2. Chi tiết các thay đổi trong phiên làm việc này

### Tối ưu hóa FSM & Tránh Cache Bloat:
* `routing_fsm.go`:
  * Định nghĩa thêm hàm helper `isDynamicModel` và `getStaticTarget`.
  * Thực hiện đi tắt (short-circuit) ngay đầu hàm `resolveDynamic`: Nếu là mô hình tĩnh (như `swe.architect` hoặc mô hình không cấu hình động), Proxy trả trực tiếp mô hình đích tĩnh mà không tạo kho khóa/đọc/ghi cache. Việc này giúp tránh phình to bộ nhớ cache (cache bloat).
  * Loại bỏ hàm `isStaticMapping` cũ và đơn giản hóa các trạng thái chuyển đổi của FSM.
* `router.go`:
  * Loại bỏ hoàn toàn hàm chết `mapDecisionToModel` không còn sử dụng.

### Cập nhật Nhật ký giải thích quyết định ("Vì [...] Nên [...]"):
* `routing_fsm.go`:
  * Thay thế các log cache cũ bằng các dòng log giải thích tường tận nguyên nhân lựa chọn model bằng tiếng Việt:
    * Static Model: `Vì model=[...] là cấu hình tĩnh nên chọn [...]`
    * Cache Hit: `Vì tìm thấy trong cache nên chọn model=[...] -> [...]`
    * LLM Router Decision: `Vì model=[...] được LLM Router phân loại là [...] nên chọn [...]`
    * Keyword Match: `Vì model=[...] khớp từ khóa '[...]' nên chọn [...]`
    * Fallback Rule: `Vì model=[...] kích hoạt luật fallback nên chọn [...]`

### Tích hợp biến môi trường `USE_LLM_ROUTER`:
* `config_loader.go`:
  * Cho phép đọc biến môi trường `USE_LLM_ROUTER` để ghi đè cấu hình: gán `true` cho `"1"/"true"` và `false` cho `"0"/"false"`. Nếu tắt LLM Router hoặc LLM Router gặp sự cố, FSM sẽ tự động rơi về cơ chế so khớp từ khóa (Keyword Match).

### Unit Tests & Tài liệu:
* `router_test.go`:
  * Thêm `TestIsDynamicModel` và `TestGetStaticTarget` kiểm thử logic phân loại mô hình tĩnh/động.
  * Thêm `TestConfigEnvOverride` kiểm thử đầy đủ các kịch bản ghi đè cấu hình qua biến môi trường.
  * Cập nhật assert log trong `TestRouterCacheMissAndHit` khớp với định dạng log tiếng Việt mới.
* Cập nhật tài liệu:
  * `docs/ROUTING_PROXY.md`: Cập nhật đặc tả hoạt động FSM có cơ chế short-circuit mô hình tĩnh.
  * `ROUTING_VERIFICATION.md`: Cập nhật mẫu log mong đợi tiếng Việt mới cho việc kiểm thử.
